# 需求文档
主面是向面企业网站开发的系统。

### 功能描述：

* 无限级栏目
* 文章管理
* 前台的模板标签
* 多模板控制
* 微信公众号支持
* 图片消息管理
* 微信菜单管理
* 全站权限控制
* 评论管理
* 收藏、点赞功能

预计开发周期2周
